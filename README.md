# Sistema Escolar (Java)

## Versión
**1.0**

## Descripción
Sistema de gestión escolar desarrollado en Java. Permite registrar estudiantes, profesores, materias, cursos, horarios y calificaciones.

## Alcance

### ✅ Qué hace
- Registrar estudiantes, profesores y materias
- Crear cursos y asignar profesores
- Matricular estudiantes en cursos
- Crear horarios de clases
- Registrar calificaciones (3 parciales + examen final)
- Calcular nota definitiva (70% parciales + 30% examen)
- Ver calificaciones por estudiante o por curso
- Estadísticas de aprobación por curso

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real
